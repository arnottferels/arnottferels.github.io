Frequently Asked Questions

General#

Why did you create this website?

This website is my digital garden, where I share updates like my works, blog, articles, and other collections. It also hosts my portfolio.

How can I get in touch with you?

You can reach me anytime through my contact page.

How can I suggest ideas, report issues, or share feedback?

I’m open to any suggestions! If you have feedback on specific posts, feel free to leave a comment anytime (for example, in the comment sections of this post or this post). For general feedback, you can reach me through my contact page.

Data & content usage#

Do you collect personal data?

This website uses a Static Site Generator (SSG), so I don’t collect personal data unless you share it directly. For more details, check the Privacy Policy.

How can I use the content on this website?

All content is owned by me unless stated otherwise. Please don’t use it for commercial purposes or modify it without permission. You’re welcome to share content using the share buttons provided on each page, if available (for example, here in this post or this post). For content in the press collection, please refer to the Attribution section for more details.

Updates & features#

How can I stay updated on new content?

To stay updated, visit the homepage for the latest posts. You can also use my RSS Feed. If you’re interested, feel free to follow me on social media for updates, announcements, and to join the discussions.

How can I navigate this website?

You can navigate the website using the navigation bar (at the top), the footer (at the bottom), or my quick links collection. You can also use any links provided on each page. For collection-specific pages, visit the collection page. If you’re using a keyboard, you can quickly search by pressing Ctrl + / on Windows or Cmd + / on macOS. Alternatively, press / to open the search function.

Is this website optimized for mobile devices?

This website was designed with a mobile-first approach (responsive web design), making it optimized for mobile use. It works well on: Chrome (iOS and Android) and Safari (iOS). On desktop, it works well on: Brave, Firefox, Chrome (Windows and macOS), and Safari (macOS). On tablets, it works well on: Safari and Chrome (iPadOS), and HUAWEI Browser (HarmonyOS).

What tools and technologies were used to build this website?

I built this website using the Astro (5.1.1) framework. For search functionality, I use the Pagefind (1.1.0) library, a static search tool. For comments, I use Giscus, a comment system powered by GitHub Discussions.
