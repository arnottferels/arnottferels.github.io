Welcome to [my personal website](https://arnottferels.github.io), currently under development. This site is designed to showcase my work, ideas, and blog. Feel free to explore and learn more about what I do.

## Features

- [Projects](https://arnottferels.github.io/projects): A collection of my professional work and projects.
- [Blog](https://arnottferels.github.io/blog): My insights, experiences, and reflections shared through blog posts.
- [About](https://arnottferels.github.io/about): Information about my background, skills, and interests.

## Development Status

The website is a _work in progress_. Here are some features currently being developed:

- Enhanced portfolio section with detailed [projects](https://arnottferels.github.io/projects) descriptions.
- Improved all content layouts (e.g., [projects](https://arnottferels.github.io/projects), [models](https://arnottferels.github.io/models), and [blog](https://arnottferels.github.io/blog)) with a bunch of tags like [keywords](https://arnottferels.github.io/keywords), [softwares](https://arnottferels.github.io/softwares), etc.
- Interactive elements for a more engaging user experience, e.g., I use [ViewTransition](https://docs.astro.build/en/guides/view-transitions/) by Astro.

## Technologies Used

- **HTML5, CSS3, & Vanilla JavaScript:** For structuring, styling, and adding interactivity to the website. I also use [SASS](https://sass-lang.com/) (SCSS) to make the CSS more maintainable and efficient. I minimize the use of JavaScript to ensure the website is efficient and performant.
- **Static Site Generator:** I use [Astro](https://astro.build/) for easy content management and deployment.
- **Hosting:** I use [GitHub Pages](https://pages.github.com/) for seamless deployment and hosting.
- **Search:** I use [Pagefind](https://pagefind.app/). You can check the implemented search functionality [here](https://arnottferels.github.io/search).
