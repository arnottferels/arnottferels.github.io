# About the Featured Image
The image above captures an intriguing perspective of an old building, taken on January 31, 2019, using my Fujifilm XT20 camera. It showcases the Railway Administration Building in Kuala Lumpur, also known as Keretapi Tanah Melayu Berhad (KTMB). To learn more about this building, you can visit its [official website](https://www.ktmb.com.my/).
